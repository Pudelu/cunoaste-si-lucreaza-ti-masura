== Description ==

Nutrella is diet and nutrition WordPress theme generally has been designed for nutritionist, dietician,and pediatrician and for the similar websites. It is also designed for the individual owner who wants to run his own business. It contains the homepage which is built with easy sections to use and the user can understand them easily. The theme also comes with the easy to use slider which is friendly with the user. The user can slide the pages very smoothly. The color picker is also available in customizer for changing the color of the theme and that is very easy to use. Its a well coded theme comaptible with the latest version of WordPress and famous plugins like wocommerce and conatact form 7. This theme is fully responsive and nicely perform in all the screen resolutions.
